隐私政策 生效日期：2026 年 1 月 1 日 本隐私政策适用于 Chrome 扩展程序 [URL Normalize Tool]（以下简称“本插件”）。 本插件主要用于开发和测试场景，用于调用插件所属公司的服务接口，对当前浏览器页面的 URL 进行标准化校验和信息查询。 1. 我们收集的数据 为实现插件的核心功能，本插件仅会收集以下最小必要数据： 当前浏览器标签页的 URL 收集方式：通过 Chrome 扩展 API 在用户主动触发插件功能时读取 收集目的：将 URL 发送至公司内部服务接口，用于分析和返回该 URL 的标准化及相关信息 本插件不会收集： 用户姓名、账号、邮箱、电话号码等个人身份信息 浏览历史（除当前激活标签页外） 键盘输入、页面内容、Cookie 或本地存储数据 2. 数据的使用方式 收集到的 URL 数据仅用于以下用途： 向公司内部 API 发起请求 获取该 URL 的标准化结果及相关业务信息 在插件界面中向用户展示结果 数据不会用于： 广告投放 用户行为分析 用户画像或追踪 与插件功能无关的任何用途 3. 数据的共享与披露 本插件会将 URL 数据仅共享给插件所属公司的内部服务系统 不会与任何第三方组织或个人共享用户数据 不会向广告平台、分析平台或数据经纪商提供任何数据 4. 数据的存储与保留 本插件本身不在本地或服务器端持久化存储 URL 数据 URL 数据仅在请求处理过程中短暂使用 插件的本地配置数据（如开关状态）仅存储在用户浏览器中（chrome.storage.local） 5. 第三方服务 本插件： 未集成任何第三方分析、广告或追踪 SDK 不使用 Google Analytics、Firebase、AdSense 等第三方服务 6. 权限说明 本插件请求的浏览器权限仅用于实现其核心功能，例如： 访问当前标签页：用于读取用户当前浏览页面的 URL 上述权限不会被用于监控用户行为或收集无关数据。 7. 联系方式 如对本隐私政策或数据处理方式有任何疑问，请联系： 📧 877865532@qq.com © 2026 徐翔宇
